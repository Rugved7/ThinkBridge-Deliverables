## JavaScript Features

### 1. Typewriter Effect
Creates an animated typing effect for the name display:
- Types "Rugved Agasti" character by character
- Continuous loop with customizable speed
- Smooth delays between cycles

### 2. Theme Toggle
Light/Dark mode switcher:
- Fixed position toggle button
- Smooth theme transitions
- Persists theme preference during session

### 3. GitHub Dialog Popup
Welcome dialog promoting GitHub profile:
- Appears 2 seconds after page load
- Styled to match site theme
- Multiple close options (button, overlay click)
- Opens GitHub profile in new tab

### 4. Contact Form Handler
Form submission with validation:
- Prevents page reload
- Logs form data to console
- Shows personalized alert message
- Auto-resets form after submission
